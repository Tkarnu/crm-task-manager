# CRM Task Manager

CRM system for managing projects and tasks.

## Features
- User registration and login
- Role-based access control
- Project management
- Task management with status
- Spring Security authentication
- PostgreSQL database
- Thymeleaf UI
- Docker support

## Technologies
- Java
- Spring Boot
- Spring Security
- Spring Data JPA
- PostgreSQL
- Thymeleaf
- Docker

## Run locally

1. Start PostgreSQL
2. Configure application.yml
3. Run:
   mvn spring-boot:run

## Docker

docker-compose up --build
